# CLAUDE.md — G_Desktop

## Identity
This is the **G_Desktop** satellite project under the Antigravity ecosystem.
Domain: `00_CORE`

Windows desktop environment organizer — configuration management, system optimization,
dotfiles, shell profiles, app launchers, and workspace automation for development workstations.

## Rules
1. Follow the governance standards defined in `docs/standards/`.
2. All scripts must be registered in `docs/library/scripts.md`.
3. Update CHANGELOG.md and DEVLOG.md with significant changes.
4. Cross-reference integrity: check `impacts:` frontmatter before finalizing edits.

## Regla de Consistencia Cruzada
Antes de finalizar cualquier edicion a un archivo que contenga frontmatter `impacts:`,
DEBES leer cada archivo listado en `impacts` y verificar que las referencias cruzadas
sigan siendo correctas. Si no lo son, corrigelas en el mismo commit/sesion.

## Project Structure
```
G_Desktop/
  .claude/          # Claude Code configuration
  .gemini/          # Gemini CLI configuration
  .codex/           # Codex CLI configuration
  .agent/           # Agent rules and workflows
  .subagents/       # Subagent manifest, dispatch, and skills
  docs/             # Documentation
    standards/      # Governance standards
    guides/         # Desktop setup guides
  scripts/          # Desktop automation scripts
  config/           # Configuration files and dotfiles
  exports/          # Exported configurations
  manifests/        # App manifests (winget, scoop, chocolatey)

```

## Domain Focus
- **Desktop Environment**: Windows shell, PowerShell profiles, terminal configs
- **App Management**: Package manifests (winget, scoop, chocolatey), app launchers
- **Workspace Automation**: Window layouts, virtual desktops, hotkeys
- **Dev Tooling**: IDE configs, Git profiles, SSH keys, PATH management
- **System Optimization**: Services, startup, registry tweaks, performance tuning
- **Dotfile Sync**: Cross-machine configuration synchronization
